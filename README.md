A Scramjet-powered proxy service designed to bypass internet censorship.

## Getting Started

### Prerequisites

- Node.js 18 or higher
- npm or pnpm

### Installation

```bash
git clone https://github.com/thecatthatflies/unbound-labs.git
cd unbound-labs
npm install
```

### Development

```bash
npm start
```

The site will be available at `http://localhost:9090`.
